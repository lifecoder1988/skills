# 成人糖尿病食养指南 Skill

## 概述
本skill基于《成人糖尿病食养指南（征求意见稿）》，提供专业的糖尿病营养指导和食疗建议。

## 触发条件
当用户咨询以下内容时，应使用本skill：
- 糖尿病饮食建议、食谱、营养指导
- 血糖控制相关的饮食问题
- 糖尿病患者的食物选择和搭配
- 不同证型（阴虚热盛、气阴两虚、阴阳两虚）的食养方案
- 食物血糖生成指数(GI)查询
- 食物交换份计算
- 各地区（东北、西北、华北、华东、华中、西南、华南）的糖尿病食谱

## 核心知识框架

### 一、食养八准则
1. **食物多样，养成和建立合理膳食习惯**
   - 五大类食物：谷薯类、蔬菜水果、动物性食物、奶豆坚果、纯能量食物
   - 三大营养素供能比：蛋白质15-20%、碳水化合物45-60%、脂肪20-35%

2. **能量适宜，控制超重肥胖和预防消瘦**
   - BMI健康范围：18.5-23.9 kg/m²
   - 超重肥胖者：减重3-5%即可获益
   - 能量需求：25-30 kcal/kg·d（轻体力活动）

3. **主食定量，优选全谷物和低GI食物**
   - 主食量：225-300克/天（生重）
   - 全谷物和杂豆：占主食1/3以上
   - 优选低GI食物（GI≤55）

4. **积极运动，改善体质和胰岛素敏感性**
   - 每周至少5天，每次30-45分钟
   - 中等强度运动占50%以上
   - 每周2次抗阻运动

5. **清淡饮食，限制饮酒，预防和延缓并发症**
   - 烹调油≤25克/天
   - 食盐≤5克/天
   - 不推荐饮酒

6. **食养有道，合理选择应用食药物质**
   - 阴虚热盛：桑叶、决明子、莲子、百合、玉竹等
   - 气阴两虚：黄芪、制黄精、桑椹、枸杞子、葛根、山药等
   - 阴阳两虚：山茱萸、肉苁蓉、山药、茯苓、肉桂等

7. **规律进餐，合理加餐，促进餐后血糖稳定**
   - 定时定量，一日三餐
   - 根据血糖监测结果合理加餐
   - 细嚼慢咽，先吃蔬菜后吃主食

8. **自我管理，定期营养咨询提高血糖控制能力**
   - 学习食物交换份使用
   - 监测血糖、体重、运动
   - 定期接受营养指导

### 二、中医证型分类

#### 1. 阴虚热盛证
**症状特征：**
- 烦渴多饮，咽干舌燥
- 多食善饥，溲赤便秘
- 舌红少津苔黄，脉滑数或弦数

**推荐食药物质：**
桑叶、决明子、莲子、百合、玉竹、金银花、菊花、铁皮石斛*

**食养方举例：**
- 石膏乌梅饮：清热泻火，生津止渴
- 地骨皮粥：清肺生津
- 天门冬粥：养阴清肺

#### 2. 气阴两虚证
**症状特征：**
- 倦怠乏力、心慌气短、盗汗自汗
- 口干舌燥，多饮多尿
- 五心烦热，大便秘结
- 腰膝酸软，舌淡或舌红暗

**推荐食药物质：**
黄芪*、制黄精、桑椹、枸杞子、葛根、山药、茯苓、鸡内金、麦芽、薏苡仁

**食养方举例：**
- 归参炖母鸡：益气养阴补血
- 猪胰汤：益气养阴
- 黄精散：益气养阴、延年益寿

#### 3. 阴阳两虚证
**症状特征：**
- 乏力自汗，形寒肢冷
- 腰膝酸软，耳轮焦干
- 多饮多尿，或浮肿少尿
- 舌淡苔白，脉沉细无力

**推荐食药物质：**
山茱萸*、肉苁蓉*、山药、茯苓、肉桂、炒紫苏籽、干姜、黑胡椒、花椒

**食养方举例：**
- 桂黄浆粥：益气温阳
- 胡桃肉炒韭菜：温补肾阳
- 金樱子煮鲫鱼汤：温补固涩

### 三、食物GI分类

#### 低GI食物（GI≤55）
**谷类及制品：**
- 整谷粒：小麦、大麦、黑麦、荞麦、黑米、莜麦、燕麦、青稞、玉米
- 谷麸：稻麸、燕麦麸、青稞麸
- 面条：强化蛋白面条、硬质小麦面条、通心面、意大利面

**薯类：**马铃薯粉条、山芋、藕粉、苕粉、豌豆粉丝

**豆类：**黄豆、绿豆、蚕豆、扁豆、鹰嘴豆、豆腐、豆腐干

**蔬菜：**山药、芋头、芦笋、菜花、西兰花、芹菜、黄瓜、茄子等

**水果：**苹果、梨、桃、李子、樱桃、葡萄、猕猴桃、柑橘、芒果

**坚果：**花生、腰果

#### 中GI食物（55<GI≤70）
**谷类：**糙米饭、小米粥、全麦面、荞麦面、黄豆挂面

**薯类：**马铃薯（煮、蒸、烤）

**水果：**哈密瓜、菠萝、水果罐头、葡萄干

#### 高GI食物（GI>70）
**谷类：**大米饭、糯米饭、速食米饭、即食大米粥、白面馒头、白面包

**薯类：**马铃薯泥、甘薯（红，煮）

**其他：**南瓜、西瓜、葡萄糖、麦芽糖、白糖、蜂蜜

### 四、食物交换份要点

**基本原则：**
- 每份提供90千卡能量
- 同类食物可以互换
- 不同类食物按份数计算

**主要食物分类：**
1. 谷薯类：23-27克（生重）
2. 蔬菜类：240-260克（综合）
3. 水果类：140-160克（综合）
4. 肉类：40-90克（视脂肪含量）
5. 大豆及制品：18-160克（视品种）
6. 坚果类：12-26克（视类型）

### 五、各地区食谱特点

#### 东北地区
- 主食：玉米、高粱米、小米、荞麦
- 特色食材：杏鲍菇、油豆角、山野菜
- 食药物质应用：藿香、百合、黑芝麻、山药、甘草

#### 西北地区
- 主食：莜麦面、荞麦面、青稞
- 特色食材：洋芋（土豆）、羊肉
- 食药物质应用：山药、枸杞子、当归、茯苓

#### 华北地区
- 主食：大麦、青稞、紫米
- 特色食材：菜心、芋头、鲈鱼
- 食药物质应用：天麻、姜、紫苏、陈皮

#### 华东地区
- 主食：糙米、荞麦、小米
- 特色食材：河鲜、水产、时令蔬菜
- 食药物质应用：赤小豆、枸杞子、桑椹

#### 华中地区
- 主食：大米、荞麦、燕麦
- 特色食材：莲藕、菱角、水生蔬菜
- 食药物质应用：紫苏、老姜、薏苡仁、山药

#### 西南地区
- 主食：玉米、荞麦、藜麦
- 特色食材：折耳根（鱼腥草）、白扁豆
- 食药物质应用：山药、白扁豆、葛根、折耳根

#### 华南地区
- 主食：大米、河粉
- 特色食材：海鲜、岭南水果
- 食药物质应用：铁皮石斛、山药、玉竹、灵芝

## 使用指南

### 问答策略

**当用户问："糖尿病患者应该怎么吃？"**
回答框架：
1. 先介绍食养八准则的核心要点
2. 强调主食定量、优选低GI食物
3. 推荐食物多样、营养均衡
4. 建议定期营养咨询

**当用户问："有什么降血糖的食物？"**
回答框架：
1. 说明食物本身不能"降"血糖，但可以帮助"控制"血糖
2. 推荐低GI食物：全谷物、杂豆类、非淀粉类蔬菜
3. 介绍食药物质的应用（根据证型）
4. 强调综合管理的重要性

**当用户问："糖尿病能吃水果吗？"**
回答框架：
1. 可以吃，但要限量和选择
2. 推荐低GI水果：苹果、梨、桃、柑橘、樱桃
3. 建议每天140-160克，分次食用
4. 最好在两餐之间作为加餐

**当用户问特定食谱：**
1. 确认用户所在地区或饮食习惯
2. 提供对应地区的食谱示例
3. 说明能量水平（1600/1800/2000 kcal）
4. 解释食物选择的原理

### 重要提醒

**必须强调的安全提示：**
1. 本指南为征求意见稿，仅供参考
2. 食养是基础，不能替代药物治疗
3. 需在医师或营养师指导下实施
4. 定期监测血糖、体重等指标
5. 个体化调整膳食方案

**不要做的事：**
1. 不要声称某种食物能"治愈"糖尿病
2. 不要建议停药或减药
3. 不要给出具体的药物剂量建议
4. 不要忽视个体差异
5. 不要推荐极端饮食法

**示例1：基础咨询**
```
用户：我刚查出糖尿病，不知道怎么吃了，能给我一些建议吗？

回答：
首先，不要过于担心。通过合理的饮食管理，很多糖尿病患者都能很好地控制血糖。根据《成人糖尿病食养指南》，我给您以下建议：

【核心原则】
1. **主食定量**：每天主食225-300克（生重），优选全谷物和杂豆，占主食1/3以上
2. **食物多样**：每天至少12种食物，每周25种以上
3. **蔬菜充足**：每天500克，深色蔬菜占一半
4. **优质蛋白**：鱼、禽、蛋、瘦肉、豆制品
5. **水果限量**：选择苹果、梨、柑橘等低GI水果，每天140-160克

【三大营养素比例】
- 碳水化合物：45-60%
- 蛋白质：15-20%
- 脂肪：20-35%

【特别提醒】
- 烹调油≤25克/天
- 食盐≤5克/天
- 规律进餐，定时定量
- 餐后适量运动
- 定期监测血糖

建议您：
1. 咨询营养师制定个性化方案
2. 学习使用食物交换份
3. 记录饮食和血糖日记
4. 3-6个月复查评估

您所在的地区是哪里？我可以给您推荐适合的食谱示例。
```

**示例2：证型食养**
```
用户：中医说我是气阴两虚型糖尿病，有什么食疗方吗？

回答：
气阴两虚证是糖尿病常见的中医证型，主要表现为倦怠乏力、气短、口干、多饮多尿等症状。

【推荐食药物质】
- 黄芪*、制黄精、桑椹
- 枸杞子、葛根、山药
- 茯苓、鸡内金、麦芽、薏苡仁

【食养方举例】

**1. 归参炖母鸡**
- 功效：益气养阴补血
- 材料：当归35克，党参35克，母鸡1只
- 做法：将母鸡处理干净，放入当归和党参，加适量水炖煮至肉烂
- 注意：去油后食用

**2. 猪胰汤**
- 功效：益气养阴
- 材料：猪胰1条，薏苡仁30克，黄芪30克，山药120克
- 做法：将黄芪、山药煎取浓汁，与猪胰、薏苡仁共煮汤

**3. 黄精散**
- 功效：益气养阴、延年益寿
- 材料：鲜黄精5000克
- 做法：洗净切片，蒸晒9次后研末，每服6克，米汤送服

【日常饮食建议】
- 多食山药、莲子、百合、银耳等养阴食材
- 适量食用枸杞子、桑椹等补益食品
- 配合低GI主食，控制总能量

【重要提醒】
1. 食药物质需在限定范围和用量内使用
2. 具体用法用量请咨询中医师
3. 食养配合药物治疗，不能替代
4. 定期复诊，根据病情调整

您需要具体的食谱示例吗？
```

回答：
根据食物血糖生成指数(GI)分类，以下是推荐的低GI主食（GI≤55）：

   - 小麦、大麦、黑麦、荞麦

2. **谷麸类**：

   - 强化蛋白面条、加鸡蛋面条
   - 硬质小麦面条、通心面、意大利面
   - 全麦面、黄豆挂面、荞麦面条

4. **杂豆类**：
   - 绿豆、赤小豆、芸豆、蚕豆
   - 豌豆、眉豆、扁豆

【低GI薯类】
- 马铃薯粉条、山芋
- 藕粉、苕粉、豌豆粉丝

【搭配建议】
1. **比例**：全谷物和杂豆占主食1/3以上
2. **总量**：225-300克/天（生重）
3. **分配**：三餐合理分配，每餐定量

【烹饪技巧】
- 避免过度烹调和糊化
- 保持谷物颗粒完整

【进餐顺序】
先吃蔬菜 → 再吃蛋白质 → 最后吃主食

【需要避免的高GI主食】（GI>70）
- 大米饭、糯米饭、白面馒头
- 即食米粥、白面包
- 马铃薯泥、炸薯条

## 技术细节

### 能量计算
- 轻体力活动：25-30 kcal/kg·d
- 中体力活动：30-35 kcal/kg·d
- 重体力活动：35-40 kcal/kg·d

示例：60kg轻体力活动女性
- 能量需求：60 × 25-30 = 1500-1800 kcal/天

### BMI计算
BMI = 体重(kg) / 身高²(m²)
- 体重过低：<18.5
- 正常：18.5-23.9
- 超重：24.0-27.9
- 肥胖：≥28.0

### 减重目标
- 速度：1-2 kg/月
- 目标：3-6个月减5-10%体重
- 收益：减重3-5%即有健康获益
